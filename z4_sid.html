1) Kiedy powstał JavaScript (podaj datę).
JavaScript został stworzony w 1995 roku.
2) Dlaczego jest to język skryptowy
Jest językiem skryptowym, ponieważ skrypty są pisane dla specjalnego środowiska wykonawczego, które może interpretować i wykonywać bez kompilacji.
3) Gdzie jest wykonywany JS (klient lub serwer)
JavaScript może być wykonywany zarówno po stronie klienta (przeglądarki) jak i serwera (np. Node.js).
4) Dlaczego jest JS to bezpieczne narzędzie
JavaScript działa w izolowanym środowisku przeglądarki (tzw. sandbox), co zapobiega bezpośredniemu dostępowi do systemu plików komputera użytkownika, co czyni go bezpieczniejszym.
5) Czy jest to język obiektowy, jeśli tak to dlaczego.
Tak, JavaScript jest językiem obiektowym, ponieważ obsługuje obiekty, prototypy i konstrukcje klasowe od ES6.
6) Wymień zalety JS:
Zaleta1: Uniwersalność - JS działa na większości przeglądarek i jest standardem do tworzenia interaktywnych stron internetowych.
Zaleta2: Łatwość nauki - w porównaniu do innych języków programowania, JavaScript jest stosunkowo prosty do nauki.
Zaleta3: Wspieranie przez społeczność - ma silną społeczność, dużą liczbę bibliotek i frameworków.
7) Framework to:
Framework to zbiór narzędzi i bibliotek ułatwiających i przyspieszających rozwój oprogramowania.
8) Do aplikacji mobilnych można użyć Framework
Tak, istnieją frameworki, takie jak React Native czy Ionic, które pozwalają na tworzenie aplikacji mobilnych z wykorzystaniem JavaScript.
9) Aplikacje responsywne to:
Aplikacje responsywne to takie, które dostosowują swój interfejs do rozmiaru ekranu urządzenia, na którym są wyświetlane.
10) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
Gdy użytkownik kliknie ten przycisk, wywołana zostanie funkcja JavaScript `WinOpen`, która otworzy nowe okno przeglądarki.
12) Opisz następującą instrukcję window.close()
Instrukcja `window.close()` zamyka obecnie otwarte okno przeglądarki, w której jest wywoływana.
